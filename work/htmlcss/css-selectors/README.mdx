# Task 1 - CSS Selectors

Select the **homeoffice programmers**.
Use the file [task.css](http://localhost/work/htmlcss/css-selectors/src/task.css) and only modify the selectors.
`#task-0` is provided to you as an example.

## General Information

Your working space is the folder `work/htmlcss/css-selectors/`.

<Highlight type="error">
  You can only change files under `work/htmlcss/css-selectors/src/`. All other
  files will be reverted for the review process.
</Highlight>

## Rules

1. You are only allowed to use **1 selector per task**:
   - Allowed: `#task-1 .covid {}`
   - Not allowed: `#task-1 .vaccine, #task-1 .sanitizer {}`
2. The selector **must** begin with the task ID eg. `#task-1`
3. You are **not allowed** to edit the HTML.
4. You are **not allowed** to use the `data-target` attriubte for your selector.
5. You are **not allowed** to use any of the following selectors:
   - `:nth-child`
   - `:nth-last-child`
   - `:nth-of-type`
   - `:nth-last-of-type`
   - any use of `+` or `~`
